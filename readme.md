# Rasa AI Chatbot

An AI-powered conversational chatbot built with **Rasa Open Source**, deployed on **Hugging Face Spaces**, and integrated with a **Streamlit frontend**.

---

# Tech Stack

* **Rasa Open Source**
* **Python 3.10**
* **Hugging Face Spaces**
* **Docker**
* **Streamlit**
* **REST API**

---

---

# Deployment

This project is deployed using:

* Hugging Face Spaces
* Docker containerization
* Public REST API access

---

# 💻 Streamlit Frontend

The chatbot frontend is built using Streamlit and communicates with the Rasa backend through REST API requests.
